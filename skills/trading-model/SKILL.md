## 零、V8.0 核心原则：三种趋势理论 + K线为王

### 三种趋势理论（V8.0 新增 ★ 简放世界观基础）

| 层级 | 内容 | 周期 | 作用 |
|:-----|:-----|:-----|:-----|
| **K线趋势** | HH+HL / LH+LL 序列 | 日~周 | 买卖点择时 |
| **经济趋势** | 产业变迁、技术革命 | 年 | 主线方向选择 |
| **时代趋势** | AI大时代、范式转移 | 10年+ | 底层信仰 |

> 「三种趋势理论解决了"为什么要坚持主线"的根本问题——时代趋势决定经济趋势，经济趋势决定K线趋势」

### 趋势判断（不用均线）

| 方法 | V5.0 旧 | V8.0 新 |
|------|--------|--------|
| 上升趋势 | MA20 > MA50 | **高点更高(HH) + 低点更高(HL)** |
| 下降趋势 | MA20 < MA50 | **高点更低(LH) + 低点更低(LL)** |
| 震荡 | 均线缠绕 | HH/LH混合 |

### ⚠️ 量化时代微操规则（V8.0 新增）

> 简放：「现在量化主导短线，传统交易规则必须调整」

| 规则 | 内容 |
|:-----|:-----|
| **前半小时不交易** | 开盘30分钟是量化表演时间：高开砸盘、低开吸筹，半小时后才回归正常逻辑 |
| **删除"破位"概念** | 量化故意破位诱空，短期跌破支撑≠趋势逆转，不需恐慌 |
| **量化连锁抛售** | 量化阈值触发连锁抛售时，短期会修复，不跟风割肉 |

### 止跌两步走

```
第一步：供应衰竭 = 连续缩量（量逐日递减）+ 量比<0.7
第二步：需求出现 = 今日收阳（close > open）+ 量略有放大
买点触发 = 第一步 and 第二步 and 上升趋势
```

### 波段量价模式（V8.0 新增）

```
上升趋势波段：始于缩量（供应衰竭）→ 终于放量（需求透支）
下降趋势波段：始于放量（恐慌出清）→ 终于缩量（无人接盘）
```

### 伪信号过滤

```
急跌后反弹 > 5% → 买点已过 → 排除
加速后不买入（近5日有单日+3%放量大阳）
```

### 加速的判定（V8.0 细化）

> 涨停不放量≠加速。加速要看**量价关系**而非涨幅数字

## 一、大盘排除法（V8.0 新增 ★ 简化大盘判断）

> 简放：「不看大盘涨跌预测，只用排除法排除两段主跌风险」

```
排除两段 = 加速后（需求透支） + 阴跌（量价齐缩持续下跌）
安全区   = 排除这两段后，不管大盘，专注主线
```

| 调整方式 | 特征 | 应对 |
|:---------|:-----|:-----|
| **急跌**（给空间） | 短期快速下跌10-20% | 等止跌信号→低吸，时间短 |
| **阴跌**（给时间） | 缓慢持续下跌 | 不急低吸，等空间到位 |

> 「急跌 >> 阴跌。急跌可快速修复，阴跌拉长时间更危险」

### 大盘定性

| 市场 | 成交量/涨跌比 | 策略 |
|------|--------|------|
| **强势** | 涨跌比>50% | 重仓，持股为主，做大波段 |
| **正常** | 涨跌比30-50% | 仓位适中，优先B4/B2 |
| **弱势** | 涨跌比<30% | 控仓，收紧止盈，屏蔽B1突破 |

### 硅基通胀 vs 碳基通缩（V8.0 新增 ★ 二元框架）

| 类型 | 代表行业 | 量价特征 | 操作 |
|:-----|:---------|:---------|:-----|
| **硅基通胀** | AI链（算力/存力/光通信/半导体） | 量价齐升 | 积极做多 |
| **碳基通缩** | 消费、地产、传统制造 | 量价齐缩 | 回避，不参与脉冲 |

> 「主线永远在量价齐升的方向中产生。PPI转正（2026.3）标志工业通缩结束，硅基通胀时代开启」

## 二、3L体系 + 主线识别

| L | 含义 | 解决的问题 |
|:---|:-----|:----------|
| **动量主线** | 跟随市场选择的主线板块 | 做什么方向 |
| **最强逻辑** | 选产业/业绩/资金逻辑最强的个股 | 做什么标的 |
| **量价择时** | 用供需格局确定买卖时机 | 什么时候做 |

### 主线内部轮动顺序（V8.0 新增）

```
光模块/光通信 → 存储/HBM → 半导体设备材料 → PCB/CCL → 封测
                    ↑ 穿插机器人/MLCC ↑
```

> 「主线轮动时不要"去弱换强"，弱者可能是下一个强者——**低吸弱、高抛强**做波段接力」

### 主线判断

**主线不会轻易切换**。判断主线见顶看**基本面出问题**，不看短期涨速。

拥挤度高≠行情见顶。真正见顶=微观高拥挤度 × **宏观基本面反转**。

### 四条选股原则

| 原则 | 说明 |
|------|------|
| **逻辑强** | 增量逻辑（量价齐升），符合时代潮流 |
| **主线强** | 属于市场已验证的主线板块 |
| **走势优** | 优先上升趋势，其次区间震荡，下降趋势回避 |
| **有催化剂** | 近期有渐进式催化事件（如中报/政策） |

### 三角思考框架

```
① 逻辑变了吗？
② 走势结构变了吗？
③ 触发了我的买卖点了吗？
```

### 四层决策框架（V8.0 新增）

```
逻辑（顶层） → 走势结构 → 买卖点 → 情绪（底层）
```

## 三、波段操作：「顺大势逆小势」

```
加速（波峰）──→ 高抛减仓 ←── 有量放宽止盈
    ↓                          ↑
回调/急跌 ──→ 恐慌衰竭 ──→ 低吸加仓 ←── 无量收紧止盈

口诀：加速高抛，回踩低吸
禁忌：不在加速后追涨，不在恐慌中割肉
```

### 「不要做最差的交易」三分法（V8.0 新增）

| 等级 | 操作 | 结果 |
|:-----|:-----|:-----|
| 🥇 最优 | 恐慌低吸 + 加速高抛 | 赚钱 |
| 🥈 中等 | 不做反（不追涨不杀跌） | 小亏可控 |
| 🥉 最差 | 高位追涨 + 低位割肉 | 大亏 |

> 「只要不做最差交易就超越 80% 的人」

### 有量/无量止盈规则（V8.0 新增 ★ 核心操作准则）

| 反弹时成交量 | 操作 |
|:------------|:-----|
| **温和放量** | 放宽止盈，等需求演绎 |
| **缺量** | 收紧止盈，需求不足及时退出 |

> 「有量就放宽止盈，没量就收紧止盈」

### 峰谷风险不对称

| 位置 | 风险 | 操作 | 原则 |
|:-----|:-----|:-----|:-----|
| **波峰**（加速区） | 高风险 | 减仓/控仓 | 加速后必调整 |
| **波谷**（恐慌/衰竭区） | 低风险 | 低吸/加仓 | 恐慌后大概率反弹 |
| **波中**（趋势中） | 中风险 | 持有 | 耐心持股 |

### 加速 vs 突破

| | 发生位置 | 后续 | 操作 |
|:--|:--------|:-----|:-----|
| **加速** | 一波上升段**末期** | 大概率调整 | 减仓 |
| **突破** | 一波上升段**初期** | 改变格局 | 可买入 |

## 四、买点体系

| 优先级 | 买点 | 类型 | 核心逻辑 | 量价特征 | 适用场景 |
|:-------|:-----|:-----|:---------|:---------|:---------|
| ★★★ | **B4 恐慌买点** | 左侧 | 「跌不动」供应出清 | 连续急跌放量阴后缩量企稳 | 调整末端 |
| ★★★ | **B2 中继买点** | 左侧 | 「跌不动」供应衰竭 | 缩量回踩支撑位获支撑 | 上升趋势回调/区底 |
| ★★☆ | **B5 衰竭买点** | 左侧 | 「跌不动」无供应 | 缩量到地量+波动收窄 | 回调/整理末期 |
| ★☆☆ | **B1 突破买点** | 右侧 | 「正在涨」需求强劲 | 放量突破关键位+站稳 | 上升形成期/区间突破 |
| ★☆☆ | **B3 反转买点** | 右侧 | 「正在涨」需求出现 | 恐慌后放量阳线反包 | 下降末端/急跌末端 |

> 口诀：**「上升趋势买衰竭（缩量），下降趋势买反转（放量）」**
> 简放最强调：**恐慌低吸**（B4）——「之前减仓的，趁着恐慌低吸，是最优操作」

## 六、风险控制

### 止损

| 买点类型 | 止损位置 |
|:---------|:---------|
| B1 突破买点 | 突破K线最低点下方 |
| B2 中继买点 | 支撑位下方1% |
| B3/B4 恐慌/反转买点 | 买入K线最低点下方 |
| B5 衰竭买点 | 整理区间低点下方 |

### 仓位管理

**仓位联动**：概率有利（波谷）→ 重仓；概率不利（波峰）→ 轻仓/空仓

**三分法仓位策略（V8.0 新增）**：
| 走势状态 | 操作 |
|:---------|:-----|
| 正常（趋势中） | 持股不动 |
| 加速区 | 逐步减仓 |
| 急跌后止跌 | 等待信号→低吸 |

**不同成本不同策略（V8.0 新增）**：
| 状态 | 策略 |
|:-----|:-----|
| 利润垫厚的 | 可持股穿越波动或适度减仓 |
| 刚追涨无利润的 | 等下次机会，波峰位置追涨不可取 |

**三个必须减仓的位置**：
1. 加速后（需求透支）
2. 需求不足/供应出现（放量滞涨/缩量滞涨）
3. 下降趋势中任何位置

### 期权对冲（V8.0 重大更新 ★）

> 简放2026年新增的核心风控手段，从概念升级为具体执行规则

**1% 规则**：
```
每次加速后，拿出总资金1%买科创50虚值看跌期权
→ 不跌：仅损失1%
→ 大跌：赚数倍，对冲主仓回撤
```

**执行核心**：
| 要点 | 规则 |
|:-----|:-----|
| **择时** | 仅加速后开仓，震荡/下跌不买 |
| **不按周/月定投** | 只在加速区操作 |
| **分散对冲** | 持仓分散时搭配沪深300/中证500期权 |
| **Theta陷阱** | 虚值期权时间价值每日损耗，震荡横盘侵蚀权利金 |
| **Vega陷阱** | 大涨后IV（隐含波动率）暴跌，指数小跌期权不赚钱 |

## 七、操作纪律

```
❌ 最差交易：加速后追涨 → 被套 → 恐慌中割肉 → 来回挨耳光 → 大亏
✅ 最优交易：加速时减仓 → 恐慌时低吸 → 反弹时持有 → 再次加速减仓
```

### 核心禁忌

| 禁忌 | 原因 |
|:-----|:-----|
| ❌ 加速中追涨 | 加速=需求透支 |
| ❌ 恐慌中割肉 | 恐慌后反弹概率最高 |
| ❌ 频繁换股追热点 | 反复挨揍 |
| ❌ 怀疑主线逻辑 | 主线不因一天波动而变 |
| ❌ 下跌时找消息面借口 | 本质是供需格局变化 |
| ❌ 开盘前半小时冲动操作 | 量化主导期，易做反（V8.0 新增） |
| ❌ 追非主线脉冲 | 后排方向反弹是脉冲，不参与（V8.0 新增） |

## 八、心态心法（V8.0 更新）

- **吸引力法则**（V8.0 新增）：你相信什么就会吸引什么。正向目标感+乐观心态是成功的底层驱动力
- **空杯心态**：放下成见，保持开放，持续学习
- **跟随思维**：市场怎么走就怎么跟，顺势而为
- **知行合一**：计划交易，交易计划，止损不犹豫
- **概率思维**：每笔交易都可能失败，做大概率事
- **忍耐**：「忍耐也是交易的一部分」
- **交易日志**（V8.0 新增）：每笔交易记录买入原因+所用买卖点+事后总结——心态修炼的具体工具
- **证实性偏见警惕**（V8.0 新增）：只看到支持自己观点的证据是认知陷阱
- **长期主义三层次**（V8.0 新增）：长期大格局 + 长期逻辑 + 长期交易体系（≠死拿套牢）
- **体系信任**（V8.0 新增）：信任体系，体系才回报你。不是你做到了才相信，而是相信了才能做到

## 十一、七大铁律（V8.0 新增第7条）

1. **下降趋势不能缩量低吸**——缩量≠见底，只等恐慌或反转
2. **交易是试错的游戏**——止损截断亏损，持盈让利润奔跑
3. **走势结构不断转换**——买卖点=供需转换点
4. **区间震荡只做两头**——区底获支撑买，区顶受阻卖
5. **加速后必调整**——加速=透支需求，先减仓再判断
6. **区分恐慌和跌破**——下降末端放量阴=恐慌=买点；区间底放量阴跌破=向下突破=卖点
7. **看得见的利空不是利空**（⭐ V8.0）——已知利空已被市场计价，真正杀伤力来自突发未知事件

## 十二、V8.0 量化扫描逻辑

### 趋势判断

```python
# K线高低点序列
上升趋势 = 最近高点 > 前高 and 最近低点 > 前低  # HH + HL
下降趋势 = 最近高点 < 前高 and 最近低点 < 前低  # LH + LL
震荡     = 不符合以上两者
```

### 大盘排除法

```python
# 排除加速后 + 阴跌 两段主跌风险
if 大盘近5日有单日急涨 > 2% and 放量(量比>1.5):
    排除原因 = "加速后"
elif 大盘连跌 > 5日 and 量逐步萎缩:
    排除原因 = "阴跌中"
else:
    安全区 = True  # 专注主线选股
```

### 止跌两步走

```python
第一步_供应衰竭 = (
    连续缩量(至少3日量<前日) or 量比 < 0.6
)
第二步_需求出现 = (
    今日收阳(close > open) and 放量(今日量 > 昨日量)
)
买点触发 = 第一步_供应衰竭 and 第二步_需求出现 and 趋势=上升
```

### B4 恐慌买点

```python
B4条件 = (
    近3日有单日急跌 > 3% or 累计 > 8% and
    今日缩量(量<昨日量) and
    今日收阳(close > open) and
    非加速区(近5日无+3%放量大阳) and
    近期无强力反弹(急跌后单日反弹≤5%) and
    开市已过30分钟  # V8.0: 量化主导期过滤
)
```

### 大盘排除法（量化版）

```python
涨跌比 = 上涨只数 / (上涨只数 + 下跌只数) × 100%
if 涨跌比 < 30: 市场=弱势 → 收紧条件，屏蔽B1突破
elif 涨跌比 < 50: 市场=正常 → 优先B4/B2
else: 市场=强势 → B4/B2正常，谨慎B1
```

### B2 中继买点

```python
B2条件 = (
    均线多头(MA5>MA20>MA60) &
    回撤2%-15%(从近15日高点) &
    缩量(当日量<20日均量×0.8) &
    属于主线板块
)
# 加分项：股价接近MA20（±3%）
```

### B1 突破买点

```python
B1条件 = (
    成交量>20日均量×1.5 &
    价格突破20日高点 &
    均线多头排列 &
    属于主线板块 &
    涨跌比>30%  # V8.0: 弱势屏蔽
)
```
