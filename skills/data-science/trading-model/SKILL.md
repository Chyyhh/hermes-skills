# 交易至简选股模型 V4 — 量价行为分析五步闭环系统

> 基于 87 篇量价分析课程文档全量提炼。核心理念：**在概率优势的位置交易**。概率优势只来自两个来源：「正在涨」（趋势跟随）和「跌不动」（供应衰竭）。

---

## 核心框架：五步闭环

```
识别结构 → 判断阶段 → 解读量价 → 波段操作 → 严格止损
```

---

## 一、交易前准备工作

### 1. 大盘定性（决定总仓位）

| 市场 | 成交量 | 特征 | 策略 |
|------|--------|------|------|
| **强势** | 日均万亿+ | 清晰主线+赚钱效应 | 重仓，持股为主，做大波段，倾向右侧 |
| **中等** | 居中 | 介于两者之间 | 仓位适中，左右结合 |
| **弱势** | 日均7000亿- | 无主线，轮动快 | 控仓，收紧止盈，多用左侧卖点，不追涨 |

### 2. 选股与构建股池（四大原则，满足越多越好）

| 原则 | 说明 |
|------|------|
| **逻辑强** | 符合时代潮流（高景气或困境反转） |
| **主线强** | 属于市场已验证的主线板块 |
| **走势优** | 优先上升趋势，其次区间震荡，下降趋势回避 |
| **有催化剂** | 近期有渐进式催化事件 |

### 3. 个股结构分析与阶段判定

> 看盘顺序：**先看结构 → 再看阶段 → 最后看量价**

**三种结构与策略：**

| 结构 | 供需 | 阶段细分 | 策略 |
|------|------|----------|------|
| **上升趋势** | 需求占优 | ①形成 ②发展 → 做多（突破/中继买点） | ③加速 ④逆转 → 准备卖出（加速/反转卖点） |
| **区间震荡** | 供需平衡 | — | 只做两头：区底供应萎缩买 → 区顶需求不足卖 |
| **下降趋势** | 供应占优 | ①形成 ②发展 → 坚决回避 | ③恐慌 ④逆转 → 等恐慌/反转买点 |

## 二、核心交易信号与操作

### 买点（波谷）

| 买点 | 核心逻辑 | 适用结构 | 量价特征 | 适用市场 | 风险收益 |
|------|----------|----------|----------|----------|----------|
| **突破买点** | 「正在涨」 | 上升形成/区间突破 | 放量阳线突破关键点，成交量明显放大 | 强势 | 失败率高，成功后空间大 |
| **中继买点** | 「跌不动」供应萎缩 | 上升回调/区底 | 回调缩量+支撑位获支撑 | 非强势 | 成功率高，风险低 |
| **反转买点** | 「正在涨」需求出现 | 下降末端/调整末端 | 供应衰竭后放量阳线反包（锤头线/阳包阴） | 弱势 | 确认需求，成功率中 |
| **恐慌买点** | 「跌不动」供应出清 | 下降末端 | 缓跌后急跌+连续放量大阴线，形成恐慌 | 非强势 | 胜率低，赔率高 |

> 📌 口诀：**「上升趋势买衰竭（缩量），下降趋势买反转（放量）」**

### 卖点（波峰）

| 卖点 | 核心逻辑 | 量价特征 | 适用市场 | 优缺点 |
|------|----------|----------|----------|--------|
| **加速卖点**（左侧） | 「涨不动」需求透支 | K线斜率变陡+连续放量大阳线 | 弱势 | 主动流动性好；容易卖飞 |
| **减速卖点**（左侧） | 「涨不动」需求不足 | K线变短+放量滞涨/缩量滞涨 | 各种 | 主动；易误判为中继 |
| **反转卖点**（右侧） | 「正在跌」供应涌现 | 放量大阴线反包（阴包阳/双顶/头肩顶） | 强势 | 不易卖飞；利润回撤大 |

> 📌 理想：**左右结合**——加速卖一半，反转卖另一半

## 三、交易思维与风控

### 止损设置

> 原则：买入前必须设置！止损点是证明买入逻辑失败的关键位置。

| 买点类型 | 止损位置 |
|----------|----------|
| 突破/反转买点 | 突破/反转K线最低点下方 |
| 中继/恐慌买点 | 买点K线最低点或关键支撑位下方 |

> ⚠️ 止损目的不是阻止亏损，而是控制亏损在最小范围，避免单笔大亏。

### 仓位管理

> 核心目的：**回避主跌段**，而非回避所有调整。

**三个必须减仓的位置：**
1. 加速后（需求透支，阳量急放）
2. 上升后需求不足/供应出现（放量滞涨/缩量滞涨）
3. 下降趋势任何位置（供过于求）

**仓位联动：** 概率有利（上升早期/波谷）→ 重仓；概率不利（加速后/下降中）→ 空仓/轻仓

### 「吃鱼」理论

```
波谷弱转强 = 鱼头（肉少刺多）
波中 = 鱼身（最好吃，耐心持股）
波峰强转弱 = 鱼尾（容易大亏，果断处理）
```

### 概率思维与等待

| 思维 | 内容 |
|------|------|
| **概率思维** | 每笔交易都可能失败。做大概率的事，严格止损防范小概率 |
| **等待** | 买入前等买点，持股中等卖点，空仓时等时机——「买点买，卖点卖」 |
| **错过思维** | 错过不亏钱，追错才亏钱。机会像公交车，过了等下一班 |

## 四、关键信号总览

| 信号 | 作用 | 操作 |
|------|------|------|
| **突破信号** | 改变格局 | 放量突破前高+站稳 → 买入 |
| **反转信号** | 逆转格局 | 吞没线/锤头/墓碑线 → 买入/卖出 |
| **衰竭信号** | 预示变盘 | 需求透支→左侧减仓；供应透支→准备买入 |
| **中继信号** | 维持格局 | 缩量回踩→买点；缩量反弹→等待 |
| **平衡信号** | 无方向 | 缩量窄幅→不操作 |

## 五、交易前五步程序

```
① 定性：上升 / 震荡 / 下降？
② 定位：波峰 / 波中 / 波谷？
③ 定信号：突破 / 反转 / 衰竭 / 中继 / 平衡？
④ 定时：是否在关键点出现关键信号？
⑤ 执行：买入(B1-B5) / 卖出(S1-S5) / 过滤(F1-F6)
```

### 买入规则 B1-B5

| 编号 | 名称 | 触发条件 | 止损 |
|------|------|---------|------|
| **B1** | 突破买点 | 震荡→放量突破前高+站稳 | 区间顶下方1% |
| **B2** | 中继买点 | 上升趋势缩量回踩关键支撑 | 支撑下方1% |
| **B3** | 反转买点 | 恐慌后放量阳线反包 | 阳线最低下方1% |
| **B4** | 反趋势买点 | 区间底突破失败+滞跌折回 | 区间底下方1% |
| **B5** | 衰竭买点 | 回调/整理中缩量到地量+波动收窄，**排除涨停板/一字板** | 最低点下方1% |

### 卖出规则 S1-S5

| 编号 | 名称 | 触发条件 | 操作 |
|------|------|---------|------|
| **S1** | 加速卖点 | 加速放量大阳（需求透支） | 减仓/清仓 |
| **S2** | 反转卖点 | 吞没线/墓碑线/反包阴线 | 减仓 |
| **S3** | 量缩卖点 | 波峰量<前波峰量+滞涨 | 减仓 |
| **S4** | 区间顶卖点 | 到区间顶+滞涨/放量滞涨 | 卖出 |
| **S5** | 破位卖点 | 放量长阴跌破关键支撑 | 清仓 |

### 过滤规则 F1-F6

| 编号 | 规则 |
|------|------|
| **F1** | 加速中不买入（加速无买点） |
| **F2** | 下降趋势不因缩量抄底（缩量≠见底） |
| **F3** | 非关键点/关键信号不交易 |
| **F4** | 充分定价后不追涨（天量天价） |
| **F5** | 下降趋势不做多 |
| **F6** | 弱势市场不追突破（假突破多） |

## 六、结构拆分六大铁律

1. **下降趋势不能缩量低吸**——缩量≠见底，可能阴跌很久。只等恐慌或反转
2. **交易是试错的游戏**——止损截断亏损，持盈让利润奔跑。优先低吸买点（止损近），谨慎突破买点（止损远）
3. **走势结构不断转换**——没有永远上升/下降段。买点=供不应求，卖点=供过于求
4. **区间震荡只做两头**——区底获支撑+供应萎缩→买入；区顶受阻→卖出
5. **加速后必调整**——加速=透支需求。先减仓再判断
6. **区分恐慌和跌破**——下降末端放量阴=恐慌=买点；区间底部放量阴跌破=向下突破=**卖点**

## 七、关键陷阱：涨停板缩量 ≠ 供应衰竭

> ⚠️ **一字板/涨停板的缩量是人为的，不是供应衰竭。**
> 
> 涨停板（含一字板）的供需原理：买方意愿极强，但卖方无人愿意卖出，导致成交量极低。这是**极端需求**信号，跟供应衰竭完全相反。
> 
> 供应衰竭（真正的 B5）必须满足：缩量发生在**回调/整理**中，股价不创新高。
> 
> **判断方法：**
> - 一字板：O=H=L=C，涨幅≈涨停板 → 排除
> - 封板涨停：收盘=最高价，涨幅≥涨停板 → 排除
> - 真正的衰竭缩量：股价横盘或小幅回调中逐渐缩量，无涨停板

## 九、模型应用流程图

```
大盘定性 → 市场强弱？
  ├─ 强势/中等 → 选股与构建股池
  └─ 弱势 → 控制仓位，收紧止盈
        ↓
  个股结构分析 → 哪种结构？
  ├─ 上升趋势 → 判断阶段
  │   ├─ 形成/发展 → 寻找突破/中继买点
  │   └─ 加速/逆转 → 准备加速/反转卖点
  ├─ 区间震荡 → 关注两头
  │   └─ 区底支撑买点 / 区顶突破买点
  └─ 下降趋势 → 非恐慌或反转不动
      └─ 出现恐慌/反转信号 → 轻仓参与
        ↓
  买入同时设置止损 → 持有等待卖点
        ↓
  出现哪种卖点？
  ├─ 加速/需求不足(左侧) → 减仓
  └─ 反转/破位(右侧) → 清仓
        ↓
  复盘总结，等待下一轮
```

---

> 一句话核心：**在供应衰竭的位置（低位/地量），寻找需求进场的证据（放量阳线/锤头/突破），避开需求透支的位置（高位/天量/墓碑线）。只在关键点出现关键信号时出手。**
