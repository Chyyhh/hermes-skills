# A500 指数数据下载记录（2026-06-09）

## 会话摘要

用户要求下载中证A500成分股过去5年的日线成交数据。

## 第一轮尝试（失败）

- 使用 `ak.index_stock_cons(symbol="000510")`（新浪财经数据源）
- 返回500行数据，但仅371个唯一股票代码
- 108只股票在同一天（2024-09-23）重复出现2-3次
- 数据库只存了371只

## 用户纠正

用户指出："A500成分股应该就有500只"

## 排查与修复

1. 使用 `ak.index_stock_info()` 查看指数列表 → 中证指数官网显示样本数量=500
2. 改用 `ak.index_stock_cons_csindex(symbol="000510")`（中证指数官网接口）
3. 返回500行×500唯一，包含中文/英文名、交易所、日期
4. 比对差异：129只股票缺失
5. 分批补缺 → 42+43+44 = 129只 → 全部成功

## 最终结果

- 500/500 只成分股 ✅
- 760,662 条日线记录
- 2020-01-02 → 2026-06-08（5.5年）
- 109 MB SQLite

## 数据验证发现

- 次新股（<5年上市）：海光信息(688041)、华大智造(688114)、联影医疗(688271)等 → 数据行数约700-900行，正常
- 停牌股：中直股份(600038)交易数据止于2022-06，正常
- 无空值/零值异常
- 571次>20%波动 → 新股首日/复牌首日放开涨跌停，正常
