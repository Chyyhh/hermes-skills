# 数据源对比：CSINDEX vs Sina（A500成分股）

## 问题发现

2026-06-09 第一次下载 A500 数据时，使用 `ak.index_stock_cons(symbol="000510")` 返回了500行数据，
但去重后只有 **371个唯一股票代码**，缺口129只。

## 原因

`index_stock_cons()` 来自新浪财经，对 A500(000510) 指数返回了包含历史调仓记录的完整日志。
同只股票因不同日期被纳入指数而出现多行重复：

```
代码     日期1         日期2         日期3
600519  2024-09-23    2025-06-13    2026-01-05
```

导致 row count = 500 但 unique code count = 371。

## 正确方案

使用中证指数官网接口：
```python
cons = ak.index_stock_cons_csindex(symbol="000510")
len(cons)                  # 500
len(cons['成分券代码'].unique())  # 500 ✅
```

`index_stock_cons_csindex()` 返回当前最新成分股快照，无历史重复。

## 对其他指数的影响

| 指数 | CSINDEX(官方) | Sina(新浪) |
|:----|:-------------:|:----------:|
| 中证A500 (000510) | 500只 ✅ | 371只 ❌ |
| 沪深300 (000300) | 300只 ✅ | 需验证 |
| 中证500 (000905) | 500只 ✅ | 需验证 |

**优先使用 CSINDEX** 接口获取所有中证指数成分股。
