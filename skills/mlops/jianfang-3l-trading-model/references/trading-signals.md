# 3L体系量化信号规则

> 基于交易者简放3L体系的量化实现

## 买入信号

| 信号 | 代码 | 条件 | 优先级 |
|:----|:----:|:------|:-----:|
| 突破买入 | BUY_BREAK | 量比>1.5 + 价格创20日新高 | ⭐⭐⭐ |
| 缩量回踩 | BUY_PULLBACK | 量比<0.85 + 价格距MA20<3% | ⭐⭐⭐ |
| 多头启动 | BUY_TREND | MA5>MA10>MA20刚形成 + 量比1.0-1.3 | ⭐⭐ |

### 缩量回踩买点（最常用）

```
条件 = (
    MA5 > MA20 > MA60              # 均线多头排列
    AND 从近15日高点回撤 2%~15%    # 健康回调
    AND 量比 < 0.85                # 明显缩量
    AND 现价 > MA20 * 0.97         # 未跌破MA20
    AND 现价 > MA60                # 仍在长期趋势上
)
```

**最佳信号**：价格距MA20 < 3% + 量比最小 → 分数最高

## 卖出信号

| 信号 | 代码 | 条件 | 优先级 |
|:----|:----:|:------|:-----:|
| 跌破支撑 | SELL_MA20_BREAK | 收盘跌破MA20 | 强制 |
| 放量滞涨 | SELL_VOL_SATURATION | 量比>2.0 + 价格不涨 | 强制 |
| 均线死叉 | SELL_DEATH_CROSS | MA5跌破MA10 | 参考 |

## 分数计算

信号评分（0-100分）：
- **动量主线** (0-30分)：行业板块排名前3=30, 前10=20, 其他=10
- **最强逻辑** (0-30分)：市值>500亿=15, 业绩增速>20%=15
- **量价择时** (0-40分)：买点类型(突破=20, 回踩=40, 启动=30) + 量比匹配度(20分)

总分≥70分 → 强烈买入；50-69分 → 可关注；<50分 → 等待
